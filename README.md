# Vehicle Management System (C)

A **console-based Vehicle Management System** created using the **C programming language**.  
This project allows users to **add, search, view, delete vehicle records** and **update login credentials** through a menu-driven interface.

--

##  Features

###  Authentication System
- Login with username & password
- Default credentials:
  - **Username:** Team
  - **Password:** Team
- Option to update username and password

###  Vehicle Management
- Add new vehicle details
- Search vehicle by name
- View all registered vehicles
- Delete vehicle by ID

###  Input Validation
- Vehicle name validation
- Manufacturer name validation
- Manufacture date validation (including leap year check)

###  Menu Driven Interface
- Simple and user-friendly console UI
- Clear navigation between options
